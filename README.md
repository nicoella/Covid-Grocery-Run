# Covid Grocery Run
A short game on staying safe during the pandemic on a grocery run.
Devpost: https://devpost.com/software/grocery-run-covid-19-edition

## Usage
Requires Java and Processing (https://processing.org/download/) to run.

Clone the repository:
```
$ git clone https://github.com/nicoella/Covid-Grocery-Run
$ cd Covid-Grocery-Run
```

Run the `covidgame.pde` file using the Processing application.
